# Halloween Crows

Halloween crows electronics project

## Setup

**Prerequisites**

- devbox
- direnv

- `devbox install`

## Parts

- [Seeed Studio XIAO ESP32-C3](https://www.seeedstudio.com/Seeed-XIAO-ESP32C3-p-5431.html)
